二分法查找 (Binary Search)

发表于 2017-03-14   |  

算法原理

二分法查找主要是用来解决”在一堆数中找出指定数”的问题, 而想用二分法查找, 这”一堆数”必须有一下特征:

  • 储存在数组中
  • 有序排列

二分法其实就是使用分治法进行折半查找. 查找的时间复杂度为O(logⁿ)也让其成为效率较高的一种算法. 不过它的缺陷也是明显的, 就是在它的限定之上:

  • 必须有序, 我们很难保证我们的数组都是有序的. 当然可以在构建数组的时候进行排序.
  • 第一个问题可能解决了, 可是又落到了第二个瓶颈上: 它必须是数组. 如果是用链表存储的, 就不能使用二分法查找了

数组读取效率是O(1), 可是它的插入和删除某个元素的效率却是O(n), 因而导致构建有序数组变成低效的事情.

利用分治法, 可以将折半查找分成四个步骤:

  1. 确定该区间的中间位置k
  2. 查找值与array[k]进行比较. 若相等则直接返回k, 否则重新确定查找区域继续查找.
  3. 这里假设数组为递增的(当然, 递减也可以, 只要是有序的), 若array[k] > 目标值, 则array[k+1, k+2, k+3...., high] > 目标值, 则查找区间为array[low,..., k-3, k-2, k-1]. 若array[k] < 目标值, 则array[low,..., k-3, k-2, k-1] < 目标值, 则查找区间为array[k+1, k+2, k+3...., high].
  4. 对新的查找区间重复步骤1, 2, 3直到查找成功为止.
