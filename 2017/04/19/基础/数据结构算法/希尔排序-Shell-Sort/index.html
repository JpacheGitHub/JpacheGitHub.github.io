希尔排序 Shell Sort

发表于 2017-04-19   |  

算法原理

希尔排序(Shell Sort)由D.L.Shell于1959年提出的一种排序算法, 在这之前排序算法的时间复杂度基本都是O[n²], 希尔排序算法是突破这个时间复杂度的第一批算法之一.

希尔排序是一种分组进行的排序, 却并不是随便分组后各自排序, 而是将相隔某个”增量”的数据组成一个子序列, 对子序列进行排序后, 使原有序列达到基本有序, 在序列基本有序的条件下, 排序的效率也就相应的提高了, 这里的”增量”的选取就非常关键了, 但是最后的增量值必须等于1.

可以将希尔排序分为以下步骤:

  1. 确定”增量”选取方式.
  2. 确定”增量”.
  3. 从下标为”增量”的数据开始, 对相隔”增量”的数据组成的子序列进行排序.
  4. 向后移动一位, 继续对相隔”增量”的数据组成的子序列进行排序.
  5. 重复步骤2, 步骤3, 步骤4, 直到”增量”值小于1时为止.

其实看到这里, 应该发现了希尔排序就是对插入排序的优化, 插入排序的效率在某些时候是很高, 比如数据本身就是基本有序的, 或者数据记录比较少的时候, 插入排序的优势比较明显, 但这两个条件还是比较苛刻的, 现实中, 数据少或者数据本身就是基本有序的都属于特殊情况, 希尔排序对插入排序改进后可以增加效率.
