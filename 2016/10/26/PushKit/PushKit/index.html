PushKit

对于VoIP App, VoIP可以减少电池电量的消耗, 并且通过注册和接收通知的方式来提供更好的体验.

Overview

PushKit框架提供的类用来使你的App接收远程推送, 推送可以有两种类型: standard 和 VoIP. Standard 推送可以提供像在 iOS 早期的版本中的推送. VoIP 推送除了 standard 推送的基本功能之外, 还提供了在一条推送显示给用户之前, 唤醒 App 来处理事件的功能.

Symbols

Classes

PKPushCredentials

PKPushCredentials 对象对推送授权 token 和推送类型进行了封装.

PKPushPayload

PKPushPayload 类包含了收到的远程推送的内容和信息.

PKPushRegistry

PKPushRegistry 对象用来请求发送推送和处理接收这些推送, 一个 push registry 对象管理一类确定类型的推送, 例如 VoIP app 需要用到的高优先级的推送. PushKit 在推送即将到达的时候唤醒你的应用, 并直接通知给请求推送服务的 push registry 对象.

Protocols

PKPushRegistryDelegate

PKPushRegistryDelegate 定义了 push registry 对象必须实现的方法. 这些方法将会在 token 失效时, 需要更新 token 时和收到远程推送时被调用.
